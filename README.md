# Outdoor-Exposure-and-Malaria-Positivity-Research-Project
This readme file provides information on the research project titled "Outdoor Exposure Pathways and Malaria Risk in Urban Settings". 
The project aims to assess the relationship between outdoor activities and malaria positivity in urban areas, while examining the contextual influence of socio-economic and demographic characteristics and the role of malaria preventive measures in shaping residual transmission.
